commd
=========
项目目录结构如下：




--commd  前端公共模块
    |----src
          |----component（组件库）
                    |------checkbox(多选框)
                    |------select(下拉框)
                    |------search(搜索框)
                    |------...
          |----libs（工具函数）
                    |------ajax(Ajax请求封装)
                    |------date(日期相关工具函数)
                    |------others(其他工具函数)
                    |------...
          |----view（视图）
          |----theme（主题库）
                    |------blue(蓝色)
                    |------yellow(黄色)
                    |------theme.scss(主题入口文件)
                    |------...
